# Apps
The Apps folder contains three pages of the application:
  - `home.py`: The home page of the application.
    - choropleth map
    - vaccination charts
      - vaccination rate
      - fully vaccinated people
      - vaccinated people
    - vaccine secured charts
      - population
      - secured vaccines
  - `vaccination_time.py`
    - multiselect tags
    - vaccination rate over time
  - `data.py`
    - data source name
    - data link
    - data frames
